Install

Parslet is at version 1.1.1.

Rubygems

 
        gem install parslet

Bundler

 
        gem 'parslet', '~> 1.1'

or if you want to track the edge:

 
        gem 'parslet', :git => 'git://github.com/kschiess/parslet.git'
